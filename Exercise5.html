Justify-items: justify-items is a property applied on the grid container. It aligns all the grid items within the grid container along the inline (row) axis. It sets the default alignment for all the grid items that do not have their individual alignment set using justify-self.

Justify-self: justify-self is applied on individual grid items. It aligns a particular grid item within its respective grid cell along the inline (row) axis, overriding the alignment set by justify-items.
